void nearest_neighboor ( routing  r,
site *  sites 
)

Implémentation de l'algorithme du plus proche voisin.

Paramètres
[in,out]rUn routing
[in,out]sitesUn tableau de sites dont il faut planifier la tournée
Précondition
Le tableau sites contient exactement routing_get_nb_sites(r) sites.
routing_insert_site() n'a jamais été appelée sur r.
Note
Si besoin, r a été initialisé avec la date de début de tournée.
Voir également
routing_set_departure_date()
Postcondition
r contient les dates d'arrivée pour tous les sites.
sites est trié selon l'ordre de visite défini par la tournée trouvée.
Note
nearest_neighboor() insert tous les sites du tableau sites dans le routing r.
Voir également
routing_get_arrival_date(), routing_get_delivery_date()
