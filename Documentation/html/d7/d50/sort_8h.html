Documentation des définitions de type

typedef int(* comparator)(void *, void *)

Ce type de données représente les fonctions qui comparent des éléments deux à deux.

Un comparateur définit une relation d'ordre entre éléments de même type, "castés" en : void *

comparator(first_value, second_value) doit renvoyer "vrai" si et seulement si first_value < second_value.

Le sens donné à la comparaison dépend de l'interprétation de l'utilisateur.
