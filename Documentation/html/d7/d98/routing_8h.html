site  site_to_insert 
)

Fonction qui permet d'insérer un site à la position courante dans une tournée.

Paramètres
__routingUn routing
site_to_insertLe site à placer à la position courante dans la tournée
Postcondition
La date d'arrivée au site inséré à la position courante est calculée.
La position courante d'insertion de job est augmentée de 1.
Note
Pour spécifier la date de début de tournée, utiliser au préalable : routing_set_departure_date()
Avertissement
Cette fonction doit être appelée exactement routing_get_nb_sites(__routing) fois !
Pas de retour en arrière : l'insertion d'un site est définitive ! (Ne peut pas être annulée)
Voir également
routing_get_current_position()

site routing_get_previous_site ( routing  __routing)

Fonction qui retourne le site visité à l'étape précédente lors d'une tournée.

Paramètres
__routingUn routing
Renvoie
Le site visité à l'étape précédente lors de la tournée
Note
Si la tournée n'a pas encore commencé ( routing_get_current_position (__routing) == 1 ), le site retourné est celui qui correspond à l'usine dans l'instance à laquelle le routing est associé.
Avertissement
Si la tournée est terminée ( routing_get_current_position (__routing) > routing_get_nb_sites (__routing) ), la fonction renvoie NULL .
