Documentation des macros

#define OUTPUT_SUCCESS   0

Cette macro indique que l'écriture d'une solution a réussi.

Lorsque write_solution() renvoie cette valeur, cela signifie que le fichier a bien été écrit.

Voir également

#define OUTPUT_ERROR   1

Cette macro indique que l'écriture d'une solution a échoué.

Lorsque write_solution() renvoie cette valeur, cela signifie que le fichier n'a pas pas pu être ouvert ou créé.
