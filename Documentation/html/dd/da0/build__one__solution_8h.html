Fournit une fonction qui génère une solution à partir de groupes : build_one_solution()

Documentation des fonctions

void build_one_solution ( solution  sol)

Fonction qui génère une solution à partir de groupes.

Paramètres
[in,out]solUne solution
Précondition
sol a été créée avec solution_new().
Aucune méthode de flowshop.h n'a été appelée sur les floswhops de sol
Aucune méthode de routing.h n'a été appelée sur les routings de sol
Postcondition
Les tableaux de jobs par groupe sont triés selon l'ordonnancement trouvé par neh() au sein du groupe.
Les tableaux de flowshops par groupe sont complets : les fonctions mentionnées ci-après sont utilisables.
Voir également
flowshop_get_duration(), flowshop_get_date()
Postcondition
Les tableaux de sites par groupe sont triés selon la tournée trouvée par nearest_neighboor() au sein du groupe.
Les tableaux de routings par groupe sont complets : les fonctions mentionnées ci-après sont utilisables.
Voir également
routing_get_delivery_date(), routing_get_arrival_date()
Postcondition
La fonction solution_get_tardiness() peut être appelée sur sol.
Note
Les ordonnancements sont trouvés grâce à la fonction neh().
Les tournées sont trouvées grâce à la fonction nearest_neighboor().
