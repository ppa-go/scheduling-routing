Documentation des fonctions

void neh ( flowshop  fs,
job *  jobs 
)

Implémentation de l'algorithme de NEH.

Paramètres
[in,out]fsUn flowshop
[in,out]jobsUn tableau de jobs à ordonnancer
Précondition
Le tableau jobs contient exactement flowshop_get_nb_jobs(fs) jobs.
Chaque job du tableau jobs vérifie : job_get_nb_machines(__job) == flowshop_get_nb_machines(fs)
flowshop_insert_job() n'a jamais été appelée sur fs.
Note
Si besoin, fs a été initialisé avec les dates de disponibilité au plus tôt de chaque machine.
Voir également
flowshop_set_availability_date()
Postcondition
fs contient les dates de fin de fabrication de tous les jobs sur toutes les machines.
jobs est trié selon l'ordre de passage défini par l'ordonnancement trouvé.
Note
neh() insert tous les jobs du tableau jobs dans le flowshop fs.
Voir également
flowshop_get_duration(), flowshop_get_date()
