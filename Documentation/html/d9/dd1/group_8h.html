group group_new ( unsigned int  nb_elements,
unsigned int *  element_to_group 
)

Fonction qui crée un nouveau group.

Paramètres
nb_elementsLe nombre d'éléments répartis en différents groupes
element_to_groupLe tableau des éléments répartis en différents groupes
Renvoie
Un nouveau group
Avertissement
Après utilisation, libérer les ressources en appelant : group_delete()
Note
La numérotation des groupes est normalisée :
  • Aucun groupe n'est vide.
  • Il n'y a aucun trou dans la numérotation.

Exemple : Soit 5 éléments répartis dans les groupes suivants :

const unsigned int element_to_group[] = {4, 1, 3, 4, 3};

La normalisation des groupes donne le résultat suivant :

  • Nombre de groupes = 3
  • Eléments du groupe 1 : 2
  • Eléments du groupe 2 : 3, 5
  • Eléments du groupe 3 : 1, 4

Puisque les groupes 2 et 5 sont vides, ils n'apparaissent plus. La nouvelle numérotation est sans trou.
