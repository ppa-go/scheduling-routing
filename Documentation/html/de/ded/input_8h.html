Documentation des macros

#define INPUT_SUCCESS   0

Cette macro indique que la lecture d'une instance a réussi.

Lorsque read_instance() renvoie cette valeur, cela signifie que le fichier a été correctement ouvert, et que la syntaxe du fichier est correcte.

#define INPUT_ERROR_OPEN   1

Cette macro indique que la lecture d'une instance a échoué.

Lorsque read_instance() renvoie cette valeur, cela signifie que le fichier n'a pas été correctement ouvert.

Voir également
read_instance()
#define INPUT_ERROR_SYNTAX   2

Cette macro indique que la lecture d'une instance a échoué.

Lorsque read_instance() renvoie cette valeur, cela signifie que le fichier a été correctement ouvert, mais que la syntaxe du fichier n'est pas correcte.
